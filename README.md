## Nanojsx Hooks

Experimental hooks with [nanojsx](https://nanojsx.io)

> This is experimental hooks.

### Hooks

- useState
- useEffect
- useReducer
- useMemo
- useCallback
- useRef
- useContext

### Usage

```bash
git clone https://github.com/herudi/nanojsx-hooks.git

cd nanojsx-hooks

deno task dev
```

> Hot-reloading support. just edit file in src folders.
